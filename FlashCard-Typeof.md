# TypeOf

## Example 1

```js
console.log(typeof []);
console.log(typeof {});
console.log(typeof null);
console.log(typeof NaN);
```

### Answer
```js
console.log(typeof []);     // object
console.log(typeof {});     // object
console.log(typeof null);   // object
console.log(typeof NaN);    // number
```
